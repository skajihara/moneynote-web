アプリに関するご質問・ご要望・不具合報告を管理者へ送信できます。

## 送信手順

1. サイドバーの「お問い合わせ」をクリックします。
2. **件名**と**本文**を入力します。
3. 「送信する」ボタンをクリックします。
4. 送信完了のトースト通知が表示されます。

## 入力項目

| 項目 | 必須 | 上限 |
|---|---|---|
| 件名 | ✓ | 100文字 |
| 本文 | ✓ | 2000文字 |

- 本文欄の右上に現在の文字数が表示されます（例: `128 / 2000`）。

## 返信について

管理者からの返信は、アカウントに登録されたメールアドレス宛に届きます。

## 注意事項

- 送信には**ログインが必要**です。
- 同一アカウントからの送信は**1時間に5回まで**に制限されています。上限を超えた場合はしばらく時間をおいてから再送してください。
