# MoneyNote Web マニュアル

MoneyNote Web は、PC の大画面に最適化した家計管理アプリです。
帳簿・カテゴリ・予算・固定費を管理し、AI が支出をアドバイスします。

## 目次

| ページ | 概要 |
|---|---|
| [ダッシュボード](dashboard) | 今月の収支・グラフ・最近の明細を確認する |
| [明細・入力](transactions) | 収支明細の入力・編集・カレンダー表示 |
| [予算・レポート](reports) | 予算管理と月別・年別・全期間のレポート |
| [AI分析](ai) | AI による支出診断・節約アドバイス・翌月予測 |
| [検索](search) | キーワード・カテゴリ・期間で明細を横断検索 |
| [固定費](fixed-transactions) | 家賃・サブスクなど定期的な支出の管理 |
| [CSV](csv) | 明細データのエクスポート・インポート |
| [設定](settings) | アカウント・帳簿・カテゴリ・メンバーの管理 |
| [管理者画面](admin) | ユーザー管理（システム管理者のみ） |

## 基本概念

### 帳簿

すべての操作は「帳簿」単位で行われます。
ヘッダーの帳簿セレクターで帳簿を選択してから各機能を使います。
複数の帳簿を作成したり、他のユーザーと共有することができます。

### 権限

帳簿メンバーには以下の権限が設定できます。

| 権限 | 参照 | 入力・編集 | カテゴリ・予算・固定費管理 | メンバー管理 | 帳簿削除 |
|---|---|---|---|---|---|
| VIEWER | ✓ | — | — | — | — |
| EDITOR | ✓ | ✓ | ✓ | — | — |
| ADMIN | ✓ | ✓ | ✓ | ✓ | — |
| OWNER | ✓ | ✓ | ✓ | ✓ | ✓ |
