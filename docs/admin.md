# 管理者画面

システム管理者（SYSTEM_ADMIN）のみアクセスできるユーザー管理画面です。

※スクリーンショットは後で追加

---

## ユーザー一覧

登録されている全ユーザーをテーブルで確認できます。

| 列 | 内容 |
|---|---|
| ユーザーID | ログインに使用する ID |
| ロール | USER または SYSTEM_ADMIN |
| 状態 | 有効 または 無効 |
| 登録日 | アカウント作成日 |
| 操作 | ロール変更・有効化・無効化・削除ボタン |

---

## ユーザーのフィルター

1. ロールフィルター（全員 / USER / SYSTEM_ADMIN）で絞り込みます。
2. 状態フィルター（全員 / 有効 / 無効）で絞り込みます。

---

## ユーザーの作成

1. 「ユーザーを作成」セクションで以下を入力します。
   - ユーザーID（3〜20文字、英数字とアンダースコアのみ）
   - パスワード（8文字以上、大文字・小文字・数字・記号を含む）
   - ロール（USER または SYSTEM_ADMIN）
2. 「作成」ボタンで確定します。

---

## ロールの変更

1. ユーザー行の「ロール変更」ボタンをクリックします。
2. USER と SYSTEM_ADMIN を切り替えられます。

> **注意**: 自分自身のロールは変更できません。

---

## ユーザーの有効化・無効化

1. ユーザー行の「無効化」ボタンをクリックすると、そのユーザーはログイン不可になります。
2. 「有効化」ボタンで再度ログイン可能な状態に戻せます。

> **注意**: 自分自身の無効化・SYSTEM_ADMIN の無効化はできません。

---

## ユーザーの削除

1. ユーザー行の「削除」ボタンをクリックします。
2. 確認後に物理削除されます。そのユーザーが OWNER の帳簿データもすべて削除されます。

> **注意**: SYSTEM_ADMIN ユーザーの削除・自分自身の削除はできません。

---

## 注意事項

- この画面は SYSTEM_ADMIN ロールのユーザーのみアクセスできます。
- 誤ってユーザーを削除した場合、データは復元できません。
