# 検索

複数月にまたがってキーワード・カテゴリ・期間で明細を横断検索できます。

※スクリーンショットは後で追加

---

## 検索フォーム

4つの条件を組み合わせて検索できます。すべての条件は省略可能です。

| 項目 | 説明 |
|---|---|
| キーワード | メモ欄の部分一致検索 |
| カテゴリ | 支出・収入カテゴリのプルダウン |
| 開始日 | 検索期間の開始日 |
| 終了日 | 検索期間の終了日 |

1. 条件を入力して「検索」ボタンをクリックします。
2. 条件を何も入力せずに検索すると全明細が表示されます。

---

## 検索結果

1. 検索結果件数とリストが表示されます。
2. 明細行をクリックすると右パネルに編集フォームが開きます。
3. 編集権限がある場合は内容を変更・削除できます。

---

## 注意事項

- 帳簿を選択していない場合は検索できません。
- 検索対象は選択中の帳簿のみです。
